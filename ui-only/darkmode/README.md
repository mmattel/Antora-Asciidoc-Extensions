# Dark Mode

This Antora UI extension has been created by Ryan Johnson and posted on the [Antora Zilup Tips Channel](  https://antora.zulipchat.com/#narrow/channel/282400-users/topic/dark.20mode/with/570746315).

The extension allows to add a dark mode to the UI with a toggle button.

Note that this extension has its own repository and is referenced here only. For more details including a detailed description see the [antora-dark-theme](https://github.com/the-dev-center/antora-dark-theme) repository.

Note that a [Merge Request](https://gitlab.com/antora/antora-ui-default/-/merge_requests/207) has been filed to the Antora UI Default.
